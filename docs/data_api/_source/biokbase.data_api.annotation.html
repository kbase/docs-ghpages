biokbase.data_api.annotation.genome_annotation module¶

Data API for Genome Annotation entities. This API provides methods for retrieving the Assembly and Taxon associated with an annotation, as well as retrieving individual features.

get_feature_ids(type_list=None, region_list=None, function_list=None, alias_list=None)¶

Retrieves feature ids based on filters such as feature types, regions, functional descriptions, aliases.

If any argument is None, it will not be used as a filter. If all arguments are None, all feature ids will be returned.

Parameters:
  • type_list – list<str>
  • region_list – list<dict> e.g., [{“contig_id”: str, “strand”: “+”|”-“|”?”, “start”: int, “stop”: int},...]
  • function_list – list<str>
  • alias_list – list<str>
