doekbase.data_api.taxonomy.taxon.api module¶

Data API for Taxon entities. This API provides methods for traversing taxonomic parent/child relationships, and accessing information such as NCBI taxonomic id, scientific name, scientific lineage, etc.

Represents a taxonomic unit, e.g., species.

Built to support KBaseGenomeAnnotations.Taxon and KBaseGenomes.Genome.
