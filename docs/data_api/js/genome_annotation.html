Creating a GenomeAnnotation object¶

To create a new object, instantiate GenomeAnnotation() using a configuration object as the input argument.

// access reference data (no token required)
var api_obj = GenomeAnnotation({
    ref: '1013/92/2',
    url: 'http://narrative.kbases.us',
    token: '',
    timeout: 6000
})

Genome annotation interface¶

class GenomeAnnotation(config)¶
Arguments:
  • config (object) – Configuration object. This object has the following fields:
  • ref - The object reference for the object to be accessed in the format expected by the workspace: workspace-id/object-id OR workspace-id/object-id/version, where the workspace and object can either be numeric identifiers or names. For example: 1019/4, 1019/4/1, or PrototypeReferenceGenomes/kb|g.166819.
  • url - The url for the GenomeAnnotation Service endpoint.
  • token - The KBase authorization token to be used to access the service.
