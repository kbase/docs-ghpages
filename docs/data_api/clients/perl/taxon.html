JavaScript Taxon Data API¶

The Taxon API can be used as a client of the Taxon Data API service.

Creating a Taxon object¶

To create a new object, instantiate Taxon() using a configuration object as the input argument.

// access reference data (no token required)
var api_obj = Taxon({
    ref: '1013/92/2',
    url: 'http://narrative.kbase.us',
    token: '',
    timeout: 6000
});

To use the object, call the Promise-wrapped functions and process the results accordingly.

// call function using Promise interface
api_obj.get_parent().then(
    function(parent) {
        do_something_with(parent);
});

API Reference¶

class Taxon(config)¶
Arguments:
  • config (object) – Configuration object. This object has the following fields:
  • ref - The object reference for the object to be accessed in the format expected by the workspace: workspace-id/object-id OR workspace-id/object-id/version, where the workspace and object can either be numeric identifiers or names. For example: 1019/4, 1019/4/1, or PrototypeReferenceGenomes/kb|g.166819.
  • url - The url for the GenomeAnnotation Service endpoint.
  • token - The KBase authorization token to be used to access the service.
throws ArgumentError:
 Bad input argument
get_parent()¶

Retrieve parent Taxon.

Throws:
  • ServiceException – Internal service error
  • AuthorizationException – Failed to authorize user for operation
  • AuthenticationException – Could not authenticate user
  • ObjectReferenceException – Error retrieving object for reference
  • AttributeException – Error retrieving requested attribute
  • TypeException – Input is of wrong type
Returns:

ObjectReference Reference to parent Taxon.
get_children()¶

Retrieve children Taxon.

Throws:... – same as get_parent()
Returns:list< ObjectReference > List of references to child Taxons.
get_genome_annotations()¶

Retrieve the GenomeAnnotation(s) that refer to this Taxon. If this is accessing a KBaseGenomes.Genome object, it will return an empty list (this information is not available).

Throws:... – same as get_parent()
Returns:list< ObjectReference > List of references to GenomeAnnotation objects.
get_scientific_lineage()¶

Retrieve the scientific lineage.

Throws:... – same as get_parent()
Returns:list<string> Strings for each ‘unit’ of the lineage, ordered in the usual way from Domain to Kingdom to Phylum, etc.
get_scientific_name()¶

Retrieve the scientific name.

Throws:... – same as get_parent()
Returns:string The scientific name, e.g., “Escherichia Coli K12 str. MG1655”
get_taxonomic_id()¶

Retrieve the NCBI taxonomic ID of this Taxon. For type KBaseGenomes.Genome, the source_id will be returned.

Throws:... – same as get_parent()
Returns:i32 Integer taxonomic ID.
get_kingdom()¶

Retrieve the kingdom.

Throws:... – same as get_parent()
Returns:string
get_domain()¶

Retrieve the domain.

Throws:... – same as get_parent()
Returns:string
get_genetic_code()¶

Retrieve the genetic code.

Throws:... – same as get_parent()
Returns:i32
get_aliases()¶

Retrieve the aliases.

Throws:... – same as get_parent()
Returns:list<string>
