Creating an Annotation¶

To create a new taxon, you will create an object that implements the interface defined in doekbase.data_api.annnotation.genome_annotation.api.GenomeInterface.

  • From the KBase Narrative (or within any Python program), instantiate the doekbase.data_api.annotation.genome_annotation.GenomeAnnotationAPI class. This will communicate with the back-end database directly:

    my_ann = GenomeAnnotationAPI(
    token=os.environ.get('KB_AUTH_TOKEN'),      # Authorization token
    services={'workspace_service_url': ws_url}, # Workspace Service to contact,
                                                #   e.g., ws_url='https://kbase.us/services/ws/'
    ref=ref)                                    # Object reference (see below)

  • If, instead, you are using Python or another language as a client, and running the GenomeAnnotationAPI within a Thrift server, you would use doekbase.data_api.annotation.genome_annotation.GenomeAnnotationClientAPI in Python or the equivalent in the target language.

In either case you will instantiate the class with an object reference, in the format expected by the KBase workspace: workspace-id/object-id OR workspace-id/object-id/version, where the workspace and object can either be numeric identifiers or names. For example: 1019/4, 1019/4/1, or PrototypeReferenceGenomes/kb|g.166819.

Genome annotation object interface¶

The interface used by both the client/server and local Python library APIs is shown below.

Feature description codes¶

doekbase.data_api.annotation.genome_annotation.FEATURE_DESCRIPTIONS¶

Mapping of feature type codes to descriptions of their meaning. The keys of this mapping give possible values for functions taking feature type identifiers, e.g. GenomeInterface.get_feature_types().

def proteins_to_fasta(ws_url='https://ci.kbase.us/services/ws/'):
    """Write FASTA file from a genome reference.

    Args:
        ws_url: Workspace service URL

    Returns:
        Full path to output file
    """
    ref = "PrototypeReferenceGenomes/kb|g.166819"
    # ref = "PrototypeReferenceGenomes/kb|g.3899"
    genome_annotation = GenomeAnnotationAPI(
        token=os.environ.get('KB_AUTH_TOKEN'), services={
            'workspace_service_url': ws_url}, ref=ref)

    # Get all the proteins with the Data API
    proteins = genome_annotation.get_proteins()
    # Create an output file and write to it
    outfile = '/tmp/3899_prot.fasta'
    with open(outfile, 'w') as f:
        for fasta_line in get_fasta(proteins):
            f.write(fasta_line)

    return outfile
