Search API¶

This page documents the skeleton of a search API. Real code will come after some prototyping and specific examples have been worked through.

The search APIs are broken down into the different types of data stores in the system:

  • DataStore - Data objects and their relationships
  • DataModel - Data types and their relationships

Each data store is represented with a separate subclass of the parent "SearchableData".

class data_api.search.SearchableData[source]¶

Base functionality for any searchable data source.

class data_api.search.DataStore[source]¶

Store of data objects.

class data_api.search.DataModel[source]¶

Store of data types, which together form a data model (type system).
